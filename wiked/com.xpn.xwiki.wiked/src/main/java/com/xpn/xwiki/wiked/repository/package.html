The repository API.

The API consists of general repository interface -- repository is made of spaces, which contains pages. Every container may create children.

Every repository has unique type assigned. There may be only one type assigned to every repository. Usage of two equal type codes is not allowed. There are following types defined for the core wiked:

  • project -- the type identifies the local Eclipse project repository.

Repositories are managed by IRepositoryManager. Repositories can be created and deleted using the IRepositoryManager interface.
